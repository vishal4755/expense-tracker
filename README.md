# 💰 Expense Tracker - Full Stack App

A full-stack expense tracking application built with React, Node.js, Express, and MySQL.

## 🚀 Features

- ✅ User Authentication (Register, Login, Logout)
- ✅ Forgot Password with Email OTP
- ✅ Add, View & Delete Expenses
- ✅ Category-wise Expense Tracking
- ✅ Budget Management with Progress Bar
- ✅ Budget Alert (80% warning)
- ✅ Day / Month / Year / All Time Filter
- ✅ Date-wise & Month-wise Charts
- ✅ Pie Chart Breakdown
- ✅ Export Expenses to CSV
- ✅ Dark / Light Theme Toggle
- ✅ User Profile & Change Password

## 🛠️ Tech Stack

### Frontend
- React.js (Vite)
- React Router DOM
- Recharts (Charts)
- Axios

### Backend
- Node.js
- Express.js
- MySQL
- Sequelize ORM
- JWT Authentication
- Bcrypt (Password Hashing)
- Nodemailer (Email OTP)
